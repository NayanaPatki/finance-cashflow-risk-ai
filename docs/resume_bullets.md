# Resume and Interview Bullets

## Resume Bullet

Built an end-to-end finance cash flow and risk intelligence platform using Python, FastAPI, React, and TypeScript, delivering a reconciled 13-week liquidity forecast, explainable receivables risk scoring, robust expense anomaly detection, interactive scenario analysis, validated finance data ingestion, and automated CI checks.

## Interview Talking Points

- Converted bank transactions, planned cash events, invoices, and management assumptions into one typed analytics workflow.
- Designed the forecast as an auditable opening-cash plus net-flow roll-forward with explicit weekly reconciliation tests.
- Used robust median/MAD anomaly detection because expense distributions are usually skewed and sensitive to outliers.
- Kept risk outputs explainable by exposing aging, concentration, dispute, and amount-multiple signals.
- Separated calculation services, API contracts, frontend presentation, and Power BI reporting boundaries.
- Added scenario controls that recalculate model drivers rather than applying cosmetic changes to final outputs.

## Suggested Portfolio Description

A CFO-ready decision workspace that helps finance teams anticipate liquidity pressure, prioritize collections, investigate unusual expenses, and test operating scenarios before cash risks become urgent.
